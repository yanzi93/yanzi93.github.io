transition

设置动画过渡样式。

transition: 过渡时间 延迟时间 过渡样式 过渡方式;

  • 过渡时间，延迟时间：m，ms
  • 过渡样式： all，width，height
  • 过渡方式：linear（默认），ease，ease-in，ease-out，ease-in-out，cubic-bezier（1，0，1，0）

注意：

  • 除过渡时间，剩下三个参数可以省略。
  • cubic-bezie：贝塞尔曲线，具体括号内求值可使用http://xuanfengge.com/easeing/ceaser/
  • 兼容ie9以下不支持。
  • 一定要具有初始样式。
  • transition属性的绑定位置，当绑定到:hover上时，当鼠标移开则样式直接恢复，不会在过度时间变换。

transform

设置2D变换的方式。可以设置旋转rotate，移动transalate。将transform属性设置到需要进行变换的具体标签上。

当具有多个变换方式时，将变换后面的。例如transform: translateX(-130px) rotateX(30deg);先进行旋转rotateX(30deg)再进行移动transform: translateX(-130px)。

rotate旋转

  • rotate( *deg) 正顺负逆

translate移动

  • translate(x位移量，y位移量);
  • translateX(x位移量);
  • translateY(y位移量);

单位px

scale缩放

  • scale(浮点数)

skew变形

  • skew( deg，deg)
  • skewX(*deg)
  • skewY(*deg)
