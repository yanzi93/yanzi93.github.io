git

git命令

  1. git init
  2. git add . 添加到本地缓存中
  3. git commit -m "status"
  4. git push 提交到仓库中
  5. git pull 将仓库分支加载到本地
  6. git checkout branch-name 切换分支
  7. git status 查看状态
  8. git merge origin/index-swiper 将分枝index-swiper的内容合并到主分支上，合并后使用git push 进行上传，一般我们会在分支进行编写到可直接使用稳定版本后进行合并。
  9. npm run start 启动

开发时git分支切换步骤

在进行开发时，我们通常在分支上进行组件的开发，当开发结束后，将最后版本添加到主分支上进行合并。

  1. 在git仓库中创建新的分支。
  2. 在本地中使用git pull 将分支加载到 本地。
  3. 本地使用git check branch-name 切换到分支上进行开发。
  4. 可以使用git status 查看是否切换成功。
  5. 使用git add . git commit -m "status" git push 上传到仓库分支上。

分支内容合并到主分支

当分支内容开发完毕后会将分支内容合并到主分支上。

  1. git checkout master 切换到主分支上
  2. git merge origin/branch-name branch-name为需要合并分支的名字，对分支内容进行合并
  3. git push 提交到主分支仓库
